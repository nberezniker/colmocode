# Antigravity Agent Rules — ColmoGravity

## Начало каждой сессии (ОБЯЗАТЕЛЬНО):
1. read_file("CONTEXT.md") — прочитай текущий контекст
2. НЕ читай весь codebase — только файлы нужные для задачи
3. НЕ включай в контекст: node_modules/, .next/, *.lock

## Формат изменений кода:
- Показывай diff, не полный файл
- Для изменений > 20 строк — показывай только изменённые блоки
- JSON вместо prose объяснений где возможно

## После выполнения задачи:
- Обнови CONTEXT.md секцию "Последние завершённые задачи"
- Если архитектурное решение — добавь в "Архитектурные решения"
