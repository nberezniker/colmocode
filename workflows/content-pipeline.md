Этот воркфлоу связывает генерацию контента (ColmoGravity CMO) с реальной публикацией и адаптацией под разные платформы.

### Алгоритм выполнения:

1. **Извлечение источника (Source)**
   - Прочитай последний вывод CMO-агента из `ColmoGravity` (например, из логов или артефактов).
   - Если источника нет, используй `NotebookLM report_create` для генерации базы на основе свежих данных.

2. **Адаптация (Multi-Platform Adapt)**
   Генерируй контент для целевых платформ:
   - **Telegram/Slack**: Короткий, емкий пост с эмодзи и ссылкой.
   - **LinkedIn/Medium**: Профессиональная статья с глубоким разбором.
   - **Instagram (Карусель)**: План на 5-10 слайдов с текстом для каждого.
   - **SEO Story**: Статья на 1500+ слов с ключевыми словами.

3. **Проверка качества (Tone Check)**
   - Пропусти контент через скилл `tone-of-voice` или `/apply-tone`.
   - Убедись, что ссылки рабочие и мета-теги прописаны.

4. **Планирование (Scheduling)**
   - Используй `rube` для создания напоминаний в Slack или записи в Google Календарь.
   - Если есть доступ, отправь посты в "Открытые линии" или через API соцсетей.

### Примеры использования:
- `/content-pipeline опубликуй итоги исследования AI-агентов везде`
- `/content-pipeline сделай карусель про 'змейку' в продажах`
- `/content-pipeline адаптируй отчет по ColmoGravity для LinkedIn`

### Правила:
- Обязательно: Для каждой платформы использовать уникальные "viral hooks" (крючки).
- Обязательно: Сохранять все адаптированные версии в папку `/home/user/content_package_[date]`.
