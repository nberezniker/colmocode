Воркфлоу для превращения сырых данных в структурированный RAG.

### Алгоритм выполнения:

1. **Захват данных (Ingestion)**
   - Прими URL, PDF или текст от пользователя.
   - Используй `firecrawl_scrape` для извлечения "чистого" контента.

2. **Обработка (Chunking)**
   - Разбей текст на логические части (Semantic Chunking) через `sequential_thinking`.
   - Выдели ключевые сущности и связи.

3. **Загрузка (Indexing)**
   - Добавь материалы в `NotebookLM` через `notebook_add_url`.
   - Если используется векторная БД (например, в ColmoGravity), подготовь JSON для импорта.

### Примеры использования:
- `/rag-knowledge-builder добавь это исследование в базу знаний проекта`
- `/rag-knowledge-builder создай базу по всем статьям из блога n8n`
