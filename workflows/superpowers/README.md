# 🦾 Antigravity Superpowers

Этот каталог содержит 10 Киллер-Суперспособностей Antigravity, разработанных для автоматизации топ-50 процессов разработки, маркетинга и аналитики проекта ColmoGravity.

## Принципы работы
1. **Lazy Loading**: Каждый файл воркфлоу весит ~1-2 KB и содержит только "заголовок" инструкции и триггеры. Это предотвращает засорение контекста (Context Poisoning).
2. **Parallel by Default**: Вся тяжелая работа делегируется параллельным субагентам через `invoke_subagent`.
3. **ColmoBrain First**: Перед каждым шагом делается `rag_query` для получения контекста, после завершения — `rag_ingest`.

## Как использовать
Просто вызовите соответствующую команду в чате, передав ей нужные параметры:
`/[команда] [описание задачи]`
