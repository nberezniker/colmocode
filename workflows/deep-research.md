Этот воркфлоу предназначен для исчерпывающего сбора данных по любой теме, конкурентного анализа или исследования рынков.

### Шаги выполнения:

1. **Инициализация поиска**
   Используй `firecrawl_search` для поиска по запросу.
   - Ограничение: 5-7 наиболее релевантных результатов.
   - Выдели URL-адреса, которые кажутся наиболее информативными.

2. **Глубокий скрейпинг (Scouting)**
   Для каждого выбранного URL выполни `firecrawl_scrape`.
   - Используй формат `markdown`.
   - Включи `onlyMainContent: true` для очистки от мусора.
   - Сохрани результаты в `/tmp/research_[timestamp]_[id].md`.

3. **Синтез знаний (NotebookLM)**
   Если задача требует интеграции с базой знаний:
   - Добавь полученные URL или сгенерированные файлы в NotebookLM через `notebook_add_url`.
   - Запусти `notebook_query` или `report_create` для извлечения ответов на специфические вопросы.

4. **Финальный отчет**
   Сформируй структурированный отчет в формате Markdown, используя:
   - Обзор темы
   - Ключевые инсайты
   - Конкретные цифры и факты (с цитатами)
   - Рекомендации по следующим шагам

5. **Сохранение в Артефакты**
   Если отчет финальный, сохрани его в директории артефактов текущего разговора.
