---
description: Humanize technical Russian text for developers
---

This workflow removes robotic patterns from technical Russian explanations to make them feel like they come from a real Senior Engineer.

### Steps:

1. **Strip Assistant Clichés**:
   - Remove: "Я здесь, чтобы помочь...", "Как ваш помощник...", "Важно помнить...".
   - Replace with: "Анализирую...", "Готово к деплою...", "Нашел узкое место...".

2. **Technical Fluency**:
   - Use common dev-slang appropriately (e.g., "коммичу", "рефакторю", "мержим").
   - Ensure relative human timing ("через 5 минут", "логи будут готовы сейчас").

3. **Formatting**:
   - Follow the "No period at end" rule for status lines.
   - Use ( - ) for readability in logs/steps.

4. **Directness**:
   - State the problem and solution immediately without preamble.
