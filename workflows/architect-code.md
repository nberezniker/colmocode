Этот воркфлоу превращает агента в полноценного Full-stack разработчика, способного самостоятельно проектировать, кодировать и проверять функции.

### Алгоритм выполнения:

1. **Проектирование (Sequential Thinking)**
   - Начни с `sequential_thinking` для разбора задачи.
   - Сформируй архитектурный план: какие файлы изменятся, какие новые зависимости нужны.
   - Создай `implementation_plan.md` в артефактах для прозрачности.

2. **Исследование контекста**
   - Используй `list_dir` и `view_file` для глубокого понимания существующих паттернов в проекте.
   - Проверь `GitHub` на наличие похожих Issue или PR.

3. **Реализация (Filesystem)**
   - Используй `write_to_file` или `replace_file_content` для внесения изменений.
   - Пиши чистый, типизированный код (TypeScript) с комментариями.

4. **Верификация (Puppeteer & Bash)**
   - Запусти локальные тесты или dev-сервер через `run_command`.
   - Используй `puppeteer` для визуальной проверки UI (скриншоты, клики).
   - Если тесты упали — вернись на шаг 1 (Самозалечивание).

5. **Финализация (GitHub)**
   - После успешных тестов создай новую ветку.
   - Сделай коммит и создай Pull Request через `github_create_pull_request`.

### Примеры использования:
- `/architect-code добавь темную тему для админки ColmoGravity`
- `/architect-code исправь баг с падением авторизации при пустом токене`
- `/architect-code напиши тесты на Puppeteer для главной страницы`

### Правила:
- **Безопасность**: Никогда не удаляй файлы без подтверждения через `notify_user`.
- **Качество**: Каждое изменение должно сопровождаться обновлением `walkthrough.md`.
- **Автономия**: Старайся исправить ошибки сборки самостоятельно минимум 3 раза перед алертом пользователю.
