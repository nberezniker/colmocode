Этот вокфлоу реализует философию "Vibe-coding": создание полноценного ПО через высокоуровневые промпты и итеративное проектирование.

### Алгоритм выполнения:

1. **Концептуализация (The Vibe)**
   - Получи описание идеи от пользователя.
   - Используй `sequential_thinking` для декомпозиции: "Что это?", "Для кого?", "Каков MVP?".

2. **Генерация PRD (Product Requirements Document)**
   - Используй Gemini для написания структурированного PRD.
   - Специфицируй стек (предпочтительно Next.js + Vanilla CSS + MCP).

3. **Сбор ресурсов**
   - Используй `firecrawl_scrape` для поиска референсов дизайна или кода.
   - Найди подходящие API или библиотеки.

4. **Генерация каркаса (Scaffolding)**
   - Используй `run_command` для инициализации проекта (например, `npx create-next-app`).
   - Примени базовую эстетику (Glassmorphism, Dark Mode, Vibrant Colors).

5. **Итеративная сборка**
   - Используй `/architect-code` для реализации ключевых функций.
   - Показывай прогресс через скриншоты `puppeteer`.

### Примеры использования:
- `/vibe-prototype создай дашборд для мониторинга прибыли вилл на Бали`
- `/vibe-prototype напиши калькулятор ROI для инвестиций в AI-агентов`
- `/vibe-prototype сделай MVP сервиса для автоматизации контента`

### Правила:
- **Скорость**: Фокус на функциональном прототипе, а не на идеальной архитектуре.
- **WOW-эффект**: Интерфейс должен выглядеть премиально с первого деплоя.
- **Инструменты**: Активно используй `generate_image` для создания ассетов и иконок.
